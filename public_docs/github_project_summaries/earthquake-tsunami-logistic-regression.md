# Earthquake-Tsunami Logistic Regression

## Summary

This project analyzes the relationship between earthquake characteristics and tsunami occurrence using statistical modeling. The workflow includes data cleaning, exploratory analysis, Spearman correlation analysis, logistic regression modeling, and classification evaluation.

## Project Type

R-based statistical modeling / logistic regression / binary classification.

## Skills Evidence

- R
- data cleaning
- exploratory data analysis
- Spearman correlation analysis
- logistic regression
- binary classification
- model evaluation
- AUC
- classification accuracy
- statistical interpretation
- reproducible project documentation

## Methods

- Processed earthquake-related variables such as magnitude, depth, and geographic features.
- Used correlation analysis to identify relevant predictors.
- Built a logistic regression model to estimate tsunami occurrence probability.
- Evaluated classification performance using metrics such as AUC and accuracy.
- Interpreted model results from a statistical and risk-prioritization perspective.

## Relevance to AI / Data Roles

This project provides evidence of statistical modeling, classification thinking, feature analysis, model evaluation, and structured analytical reporting. It supports applications to data analysis, business analytics, risk analysis, and entry-level machine learning roles.

## Limitations

This project should be described as statistical modeling and binary classification, not as deep learning or a production-grade disaster warning system. The model is useful for analytical demonstration and prioritization logic, but it should not be overstated as a real-world tsunami warning solution.
